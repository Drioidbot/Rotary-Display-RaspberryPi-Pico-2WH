Pico 2 W Script Menu
A feature-packed menu system for the Raspberry Pi Pico 2 W and Waveshare 0.96" LCD, built in MicroPython.
 Plug the display onto the GPIO header, flash the firmware, and you've got a pocket-sized macro pad, game console, and remote control — all in one.
